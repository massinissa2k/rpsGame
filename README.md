# rpsGame
game of Rock, Paper and Scissors

### Prerequisites
    composer 1.9.1, php 7.4

## Install
    composer install

## Run
    composer run-script serve
    //Will execute by Default 'php -S localhost:8089'
    //Browser url: http://localhost:8089/src/public/
    
## Test
    composer run-script test
